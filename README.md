# NestJS Starter Project

## Description

Starter project for [NestJS](https://docs.nestjs.com/).

## Setup

### System Information

- [Node](https://nodejs.org/en/) : v16.13.0
- [NPM](https://www.npmjs.com/) : 8.13.2
- [PostgreSQL](https://www.postgresql.org/) : 11.6

### Frameworks / Guides

- [NestJS](https://nestjs.com/)
- [TypeORM](https://typeorm.io/#/)

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

## run unit tests in watch mode
$ npm run test:watch

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Building the app

```bash
# production build
$ npm run build
```

## Utility scripts

```bash
# formatting files
$ npm run format

# check lint
$ npm run lint
```
